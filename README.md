# 🌑 구현 기능

✅ zustand를 활용한 user 상태 관리

✅ tanStack query를 활용하여 mbti 테스트 CRUD 관리

- Create: 새로운 MBTI 테스트 결과 저장
- Read: 사용자들이 작성한 모든 테스트 결과 보기
- Update: 내가 테스트한 결과의 비공개/ 공개 전환 여부 설정
- Delete: 나의 결과 삭제 기능
